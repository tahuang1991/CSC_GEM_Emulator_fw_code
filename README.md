## CSC_GEM_Emulator_fw_code  TAMU2014 branch

CSC_GEM_Emulator_fw_code is developed to read or write/send GEM/CSC data from/to PC,and to OTMB through fibers  



This version of GEMCSC emulator originated from TAMU development back to 2014 and it is only used for CSC tests. 
the original code is the master branch under: https://github.com/cms-tamu/CSC_GEM_Emulator_fw_code


### fibers in CSC_GEM_Emulator_fw_code
The optical fibers includes snap fiber1-7 to send out data to OTMB
